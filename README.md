# CN:hyqypic 浏览器插件

[ENGLISH](#en:hyqypic-browser-extension)

> 一款开源、轻量的浏览器插件，核心旨在解决网站图片难以保存，并且其他插件难以使用或需要支付或有广告的问题。

## 📖 项目简介

hyqypic 是一款面向全部用户的浏览器插件，旨在解决网站图片难以保存，并且其他插件难以使用或需要支付或有广告的问题。

**主要特点**：
- 🚀 **轻量高效**：无冗余依赖，启动迅速
- 🔒 **隐私安全**：完全开源，所有逻辑透明可审查，不上传任何用户数据
- 🎨 **极致简约**：本插件只有一个简约的右键菜单，不影响使用。

## 🔧 安装方式

### 从源码安装（开发模式）
1. 下载源码
2. 打开浏览器扩展管理页面：
   - Chrome/Edge: 地址栏输入 `chrome://extensions/` 并回车
   - Firefox: 地址栏输入 `about:debugging#/runtime/this-firefox` 并回车
3. 开启“开发者模式”（Chrome/Edge 需在右上角切换）
4. 点击“加载已解压的扩展程序”，选择项目zip文件拖入窗口即可

### 从应用商店安装
- [EDGE插件链接地址请点击此处](https://microsoftedge.microsoft.com/addons/detail/hyqy-pic/pnjedhjkemelmjfecbkpcdhiomenpcgm)

## 🚀 使用方法

1. 安装完成后，浏览器工具栏会出现插件图标
2. 右键图片、CSS图标即可选择复制图片、复制链接、下载图片、新标签打开


## 📦 依赖与环境要求

- 浏览器：Chrome 80+ / Firefox 90+ / Edge 80+
- 无需额外运行环境

本项目采用 **MIT License** 开源。

# EN:hyqypic Browser Extension

> An open-source, lightweight browser extension designed to solve the problem of difficult image saving on websites, where other extensions are either hard to use, require payment, or contain ads.

## 📖 Introduction

hyqypic is a browser extension for all users, aimed at addressing the issue that images on websites are hard to save, and other extensions are either inconvenient, paid, or ad-laden.

**Key Features**:
- 🚀 **Lightweight & Efficient**: No redundant dependencies, fast startup
- 🔒 **Privacy & Security**: Fully open-source, all logic is transparent and auditable, no user data is uploaded
- 🎨 **Minimalist Design**: Features only a simple right-click menu — clean and unobtrusive.

## 🔧 Installation

### Install from Source (Developer Mode)
1. Download the source code
2. Open your browser's extension management page:
   - Chrome/Edge: Enter `chrome://extensions/` in the address bar and press Enter
   - Firefox: Enter `about:debugging#/runtime/this-firefox` in the address bar and press Enter
3. Enable "Developer Mode" (for Chrome/Edge, toggle the switch in the top-right corner)
4. Click "Load unpacked" and select the project ZIP file, or simply drag and drop it into the window

### Install from Store
- [Click here for the EDGE add-on link](https://microsoftedge.microsoft.com/addons/detail/hyqy-pic/pnjedhjkemelmjfecbkpcdhiomenpcgm)

## 🚀 How to Use

1. After installation, the extension icon will appear in the browser toolbar
2. Right-click on any image or CSS icon to choose from: Copy Image, Copy Link, Download Image, or Open in New Tab

## 📦 Dependencies & Environment Requirements

- Browsers: Chrome 80+ / Firefox 90+ / Edge 80+
- No additional runtime environment required
